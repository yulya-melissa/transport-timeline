# Хроника транспортных изобретений России

Интерактивный историко-просветительский сайт, посвящённый эволюции транспортной науки и техники в России — от паровой телеги Кулибина до современных электромобилей.

## Стек

- Vue 3 / Nuxt 3
- Статическая генерация (SSG)
- GitHub Pages

## Запуск локально

```bash
npm install
npm run dev
