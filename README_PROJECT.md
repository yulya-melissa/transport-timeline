# 🚗 История Российского Транспорта - Интерактивный Сайт

Интерактивный проект о развитии транспорта в России с фокусом на автомобилестроение (1791–2018) и параллельные ветви развития других видов транспорта.

## ✨ Что было реализовано

### 📊 Основные данные
- **21 исторический объект** в хронологической последовательности
- **11 объектов** автомобильной линии (основной фокус)
- **10 объектов** соседних ветвей (судоходство, авиация, железные дороги, космос)
- **Временной диапазон**: 1791–2018 годы

### 🎨 Спиральная временная лента
- Плавная mini-spiral анимация при скроллинге
- Карточки появляются поэтапно со стагнированными задержками
- Альтернирующее расположение (левое-правое)
- Центральная линия-разделитель с точками

### 🔍 Интерактивные фильтры
- **По году**: клик на год фильтрует все объекты за этот год
- **По типу транспорта**: Наземный, Водный, Железнодорожный, Воздушный, Космический
- **По статусу**: Ключевой, Важный, Косвенно связан, Спорный, Проект
- **По ветке**: Автомобильная линия / Соседние ветви
- **Поиск**: по названию, изобретателю, описанию с подсказками
- **Случайный объект**: 🎲 кнопка для изучения

### 📄 Детальные страницы каждого объекта
- История создания
- Технические характеристики в таблице
- Интерактивное изображение с hover-деталями (hotspots)
- Связанные личности
- Источники информации
- Навигация к предыдущему/следующему объекту

### 🖼️ Интерактивные изображения
- Кликабельные области (hotspots) на изображениях
- Всплывающие подсказки при наведении
- Полноразмерные изображения при клике

### 💎 Дизайн
- **Минималистичный стиль** с историческим акцентом
- **Палитра**: теплые земляные тона (бежевый, кирпичный, золотой)
- **Адаптивность**: отзывчивый дизайн для мобильных и десктопов
- **Плавные переходы**: все элементы имеют smooth анимации

### 📋 Footer с информацией об источниках
- Список всех источников проекта
- Описание структуры проекта
- Информация о количестве объектов
- Контактная и авторская информация

## 🎯 Основные объекты автомобильной линии

1. **Самодвижущийся экипаж Кулибина** (1791) — педальный привод, управление, торможение
2. **Автомобиль Яковлева и Фрезе** (1896) — первый ДВС в России
3. **Электромобиль Ипполита Романова** (1899) — ранний электротранспорт
4. **Электрический омнибус Романова** (1901) — первый электробус
5. **Вездеход Пороховщикова** (1914–1915) — гусеничный движитель
6. **Аэросани ЦАГИ** (1920–1930е) — воздушный винт и лыжное основание
7. **ГАЗ-М20 "Победа"** (1946) — несущий кузов, современный автомобиль
8. **ГАЗ-21 "Волга"** (1956) — советский седан премиум-класса
9. **ВАЗ-2101 "Копейка"** (1970) — массовый народный автомобиль
10. **ВАЗ-2121 "Нива"** (1977) — первый компактный SUV с полным приводом
11. **КАМАЗ-6282** (2018) — современный электробус

## 🎨 Кастомизация

### Изменение цветов
Отредактируйте переменные в `app.vue`:
```css
:root {
  --bg: #FBF7F0;        /* фон */
  --accent: #B5443C;    /* красный акцент */
  --gold: #C4A35A;      /* золотой */
  --text-secondary: #6B5E4A;  /* серый текст */
}
```

### Добавление новых объектов
1. Отредактируйте `content/inventions/timeline.json`
2. Добавьте изображение в `public/inventions/`
3. Сервер автоматически обновит содержимое
